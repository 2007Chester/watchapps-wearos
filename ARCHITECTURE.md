# Архитектура WatchApps Wear OS

## Основные задачи

- Приём файла APK от телефона
- Проверка метаданных (hash, размер)
- Запуск установки через системный установщик
- Логирование статуса установки
- Отправка статуса обратно на Android и backend

## Взаимодействие

- с Android через Data Layer / FileTransfer
- косвенно с backend (через Android)
