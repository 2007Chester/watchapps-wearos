# WatchApps Wear OS

Приложение **WatchApps** для часов на Wear OS.

## Назначение

- Приём APK‑файлов от Android‑приложения
- Проверка целостности и хеша файлов
- Вызов системного установщика на часах
- Отправка статуса установки (успех/ошибка)
- Отображение списка доступных / установленных циферблатов

## Технологии (план)

- Kotlin
- Wear OS (Tiles, Complications — позже)
- Data Layer / FileTransfer
